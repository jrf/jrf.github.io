# \$G_delta\$_set

## Definition

A set $B \subseteq \mathbb{R}$ is called a *$G_{\delta}$ set* if it can be written as the [countable](/notes/areas/mathematics/definitions_theorems/set_theory/countable/) intersection of [open_set](/notes/areas/mathematics/definitions_theorems/topology/open_set/)s.
