# Set_function

## Definition

A *set function* is a [function](/notes/areas/mathematics/definitions_theorems/set_theory/function/) whose [domain](/notes/areas/mathematics/definitions_theorems/set_theory/domain/) is a family of subsets of some given set and that (usually) takes its values in the extended real number line $\mathbb{R} \cup \{ \pm \infty\}$, which consists of the real numbers $\mathbb{R}$ and $\pm \infty$.
