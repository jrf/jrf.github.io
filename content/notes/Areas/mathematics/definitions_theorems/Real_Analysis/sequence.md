# Sequence

## Definition

A *sequence* is a [function](/notes/areas/mathematics/definitions_theorems/set_theory/function/) whose domain is $\mathbb{N}$.

## Examples

1.  $\left. (1,1/2,1/3,1/4,\ldots) \right.$

2.  $\left. \left( \frac{1 + n}{n} \right) \right._{n = 1}^{\infty} = \left. (2/1,3/2,4/3,\ldots) \right.$

3.  $\left. \left( a_{n} \right) \right.$, where $a_{n} = 2^{n}\quad\forall n \in \mathbb{N}$

4.  $\left. \left( x_{n} \right) \right.$, where $x_{1} = 2$ and $x_{n + 1} = \frac{x_{n} + 1}{2}$
