# Supremum

## Definition

A real number $s$ is the *supremum* (a.k.a. *least [upper bound](/notes/areas/mathematics/real_analysis/definitions/upper_bound/)*) for a set $A \subseteq \mathbb{R}$ if it meets the following two criteria:

1.  $s$ is an [upper_bound](/notes/areas/mathematics/real_analysis/definitions/upper_bound/) for $A$;

2.  if $b$ is any [upper_bound](/notes/areas/mathematics/real_analysis/definitions/upper_bound/) for $A$, then $s \leq b$.
