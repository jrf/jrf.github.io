# Infimum

## Definition

A real number $i$ is the *infimum* for a set $A \subseteq \mathbb{R}$ if it meets the following criteria:

1.  $i$ is a [lower_bound](/notes/areas/mathematics/real_analysis/definitions/lower_bound/) for $A$

2.  If $l$ is any [lower_bound](/notes/areas/mathematics/real_analysis/definitions/lower_bound/) for $A$, then $i > l$
