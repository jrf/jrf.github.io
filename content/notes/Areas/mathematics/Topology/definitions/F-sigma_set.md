# \$F_sigma\$_set

## Definition

A set $A \subseteq \mathbb{R}$ is called an *$F_{\sigma}$ set* if it can be written as the [countable](/notes/areas/mathematics/set_theory/definitions/countable/) union of [closed_set](/notes/areas/mathematics/topology/definitions/closed_set/)s.
