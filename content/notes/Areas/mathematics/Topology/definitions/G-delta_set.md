# $G_\delta$ Set

## Definition

A set $B \subseteq \mathbb{R}$ is called a *$G_{\delta}$ set* if it can be written as the [countable](/notes/areas/mathematics/set_theory/definitions/countable/) intersection of [open_set](/notes/areas/mathematics/topology/definitions/open_set/)s.
